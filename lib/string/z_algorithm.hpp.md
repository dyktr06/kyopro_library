## Z Algorithm

#### 概要

文字列 $S$ のそれぞれの $i \: (0 \leq i < \lvert S\lvert)$ について、$S[1] + S[2] + ...$ と $S[i] + S[i + 1] + ...$ の最長共通接頭辞の長さ を求めます。

#### 計算量

$\mathrm{O}(\lvert S\lvert)$
